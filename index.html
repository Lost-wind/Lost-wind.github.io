HelloWorld Flask学习(01)

1.创建工程文件夹microblog
2.创建module（也称 包）文件夹app，并在其中创建 __init__.py脚本。一般把包含此脚本的文件夹称作 modeule/包）
3.在__init__.py中导入flask，并创建实例 test_app.
from flask import Flask
test_app = Flask(__name__) # __name__指向正在调用它的模块的名字（后面章节才会涉及）
from app import routes

4.在app文件夹下创建routes.py脚本.路由是应用程序实现的不同URL。在Flask中，应用程序路由的处理程序被编写为Python函数，称为视图函数。视图函数映射到一个或多个路由URL，以便Flask知道客户端请求给定URL时要执行的逻辑。
from app import test_app
@test_app.route(‘/‘) # 实例的路由属性
def index():
return ‘test!!!’

首先就导入了test_app 实例，由此可推知为什么from app import routes写在最后

5.在microblog文件夹下创建microblog.py，输入
from app import test_app

6.进入命令行，进入虚拟环境，进入microblog文件夹，输入
set FLASK_APP = microblog.py
flask run
